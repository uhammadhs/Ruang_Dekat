# UI Guidelines

## Prinsip desain

- Mobile-first.
- Background netral.
- Card besar, whitespace cukup.
- Warna tidak lebay: biru sebagai primary, hijau sebagai trust/community accent.
- Tidak memakai emoji sebagai elemen utama.
- Teks pendek, jelas, dan action-oriented.
- Desktop memakai 3 kolom: sidebar, feed, panel kanan.

## Token warna

- Background: #F8FAFC
- Text: #0F172A
- Muted: #64748B
- Primary: #2563EB
- Accent: #16A34A
- Warning: #F59E0B
- Danger: #DC2626
